# Zotero 快捷键

## 引用

- `Ctrl + Shift + A`：文中引用
- `Ctrl + Shift + C`：文末引用

## 阅读

- `Ctrl + F`：搜索
- `Alt + ←/→`：后退/前进
- `Ctrl + +/-`：放大/缩小
- `Alt + 1/2/3/4`：对应阅读时上面的四个工具（第四个是改变颜色）
- `Backspace/Delete`：删除选中的笔记
- `Esc`：退出工具或退出选中
- `Shift + 左键`：在笔记中重新选中

  > [!TIP|label:提示]
  > 比如高亮某段文字后想翻译其中一部分，就需要 `Shift + 左键` 来选中那部分，否则无法操作。

## 常见问题

### 如何更改字体大小

`编辑 > 首选项 > 高级 > 常规 > 高级设置 > 编辑器`，在其中输入 `fontsize` 即可找到 `extensions.zotero.fontSize`，双击可以更改对应的值。
